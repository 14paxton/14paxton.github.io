# Install sdkman

```bash
curl -s https://get.sdkman.io | bash 
source "$HOME/.sdkman/bin/sdkman-init.sh" 
```

# list candidates

```shell
sdk list 
```

# list versions

```shell
sdk list [program] 
```

# will use for cmd shell only

```shell
sdk use [program] [version number] 
```

# will switch default

```shell
sdk default [program][version number] 
```

# list all candidates

```bash
ls -l ~/.sdkman/candidates/java
```
